BORDER REGION INFRASTRUCTURE ANALYSIS - SUMMARY
================================================================================

Created: border_infrastructure_analysis.py
Guide: BORDER_INFRASTRUCTURE_GUIDE.md

PURPOSE
-------
Analyze charging infrastructure capacity and utilization in border NUTS2
regions, comparing scenarios with max 4 focused plots.

THE 4 PLOTS (As Requested)
---------------------------

1. CAPACITY BY SCENARIO (2×2 Grid)
   - Shows total MW installed in border regions
   - Fast vs slow charging breakdown
   - Years: 2030, 2040, 2050
   - One subplot per scenario
   File: border_infrastructure_capacity_by_scenario.png (16×12")

2. CAPACITY DELTAS (Heatmaps + Bars)
   - Top: Country × Scenario heatmaps (3 years)
   - Bottom: Total delta bars (3 years)
   - Green = more than reference, Red = less
   File: border_infrastructure_capacity_deltas.png (18×10")

3. UTILIZATION COMPARISON (3 Panels)
   - Utilization rates for 2030, 2040, 2050
   - Bar charts comparing all scenarios
   - Reference scenario highlighted
   - Shows efficiency of infrastructure
   File: border_infrastructure_utilization.png (18×6")

4. SUMMARY TABLE (Visual Table)
   - All scenarios × all years in one view
   - Fast, slow, total capacity
   - Deltas vs reference
   - Utilization rates
   File: border_infrastructure_summary_table.png (16×10")

DATA EXPORTS (3 CSV Files)
---------------------------
1. border_infrastructure_deltas.csv - Detailed delta analysis
2. border_infrastructure_all_scenarios.csv - Complete dataset
3. border_infrastructure_summary_stats.csv - Aggregate statistics

KEY FEATURES
------------
✓ Exactly 4 plots (not overwhelming like electrification analysis)
✓ Focused on border regions only (14 NUTS2 codes)
✓ Capacity in MW (fast + slow charging)
✓ Utilization rates (actual/maximum)
✓ Delta analysis vs reference scenario
✓ Years: 2030, 2040, 2050
✓ Publication-ready (300 DPI PNG)

METRICS PROVIDED
----------------
For each scenario × year × country:
- Total capacity installed (MW)
- Fast charging capacity (MW)
- Slow charging capacity (MW)
- Utilization rate (%)
- Delta vs reference (MW and %)
